# to-do-list-app

##  Build dockers
```
docker-compose build
```

### Launch dockers
```
docker-compose up
```

### Swagger
```
http://0.0.0.0:5001/docs/

Example swagger token : Bearer: jwt-token
```

### Web site
```
http://0.0.0.0:8080/

Example login : 
    Username : romane
    Password : romane 
```
